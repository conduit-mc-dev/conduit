Header 按钮状态矩阵

每种服务器状态下，Header 右侧的按钮组合

完整矩阵

状态 Start Stop Kill Delete Cancel
Stopped ✓ Start — — ✓ Delete —
Starting — — — — ✓ Cancel
Running — ✓ Stop ✓ Kill — —
Stopping — — ✓ Kill — —
Crashed ✓ Start — ✓ Kill ✓ Delete —
Installing — — — — ✓ Cancel
Downloading — — — — ✓ Cancel

按钮样式规范

主操作按钮（Primary/Start） — 透明底，绿色文字 #3fb950，绿色描边 rgba(63,185,80,0.35)。Start / Save / Create / Connect

Stop — 透明底，白色文字 #e6edf3，灰色描边 rgba(48,54,61,0.6)

次操作按钮（Secondary/Kill） — 透明底，灰色文字 #8b949e，灰色描边 rgba(33,38,45,0.6)

危险按钮（Danger/Delete） — 透明底，红色文字 #f85149，红色描边 rgba(248,81,73,0.25)

警告按钮（Warning/Cancel） — 透明底，黄色文字 #d29922，黄色描边 rgba(210,153,34,0.25)

禁用按钮（Disabled） — 透明底，暗灰文字 #484f58，灰色描边 rgba(33,38,45,0.4)

布局规则 — 主操作在最左，Kill 居中（仅 Running/Stopping/Crashed），Delete 始终最右（仅 Stopped/Crashed），Cancel 最右（仅 Starting/Installing/Downloading）
