按钮样式规范

主操作按钮（Primary） — 绿色填充 #238636，白色文字。每个状态只有一个主操作：Stopped→Start, Running→Stop, Crashed→Start

次操作按钮（Secondary） — 深灰填充 #30363d，白色文字。Kill

危险按钮（Danger） — 红色描边 #f85149，红色文字，透明背景。Delete

警告按钮（Warning） — 黄色填充 #9e6a03，白色文字。Cancel 安装/下载

禁用按钮（Disabled） — 深黑背景 #161b22，极暗文字。不可点击

布局规则 — 主操作在最左，Kill 居中（仅 Running/Stopping/Crashed），Delete 始终最右（仅 Stopped/Crashed），Cancel 最右（仅 Starting/Installing/Downloading）
